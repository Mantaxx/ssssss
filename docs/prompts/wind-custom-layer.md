# niestandardowa warstwa wiatru mapbox gl js v3

**Prompt:**

```text
Używając Mapbox GL JS v3, stwórz niestandardową warstwę (custom layer) do animacji cząsteczek wiatru. Wejściem jest siatka wektorów wiatru (prędkość i kierunek) dla zadanej wysokości. Cząsteczki poruszają się zgodnie z wektorami i zanikają po określonym czasie życia. Gęstość cząsteczek powinna wzrastać proporcjonalnie do prędkości wiatru. Zapewnij API warstwy do zmiany wysokości i czasu (timestamp), tak aby animacja aktualizowała się na żądanie.
```
